中二桥

说明：
“中二桥” 来源于一个几个月以前的设想
试图建一个 二中校友们 生活、学习、工作等经验分享的网站
而本人无任何前端设计经验
so ~~~
遵循模板 + 个人网站 + 无法交互
欢迎相关专业的大佬推倒重做

欢迎在此分享经验

  • 该网页支持 markdown 语法
    markdown语法非常适合写博客，入门简易。本人认为，若无对花里胡哨的追求，花费 5 分钟学会最简单的字体大小设置，黑点设置已够用
    若需要呈现效果，需要相关环境配置的话则需要花费一定时间
  • 接受经验分享方式：
    • 照片
      PS: 背景图案为百度搜索图案，略显草率。
    • 文章

一些展望
